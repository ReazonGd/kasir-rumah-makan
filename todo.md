## ✅ Checklist Penilaian Tugas UAS OOP (Python – Aplikasi Restoran)

- [ ] Menggunakan bahasa pemrograman Python
- [ ] Program dapat dijalankan tanpa error fatal

### Konsep OOP
- [ ] Menggunakan **kelas abstrak** (`MenuItem`)
- [ ] Mengimplementasikan **inheritance** dengan benar
- [ ] Mengimplementasikan **polymorphism** (`tampilMenu()` berbeda tiap kelas)
- [ ] Mengimplementasikan **encapsulation** (atribut private + getter/setter)

### Struktur Kelas
- [ ] Kelas `MenuItem` memiliki atribut nama, harga, kategori
- [ ] Kelas `Makanan` sebagai turunan `MenuItem`
- [ ] Kelas `Minuman` sebagai turunan `MenuItem`
- [ ] Kelas `Diskon` sebagai turunan `MenuItem`
- [ ] Kelas `Menu` untuk menyimpan daftar menu
- [ ] Kelas `Pesanan` untuk menyimpan pesanan pelanggan

### Diskon
- [ ] Diskon direpresentasikan sebagai **objek**
- [ ] Diskon dapat dipilih oleh pelanggan
- [ ] Diskon diterapkan pada perhitungan total
- [ ] Hanya satu diskon per pesanan
- [ ] Diskon ditampilkan di struk

### Fitur Program
- [ ] Menu utama interaktif
- [ ] Menambah item menu (makanan, minuman, diskon)
- [ ] Menampilkan daftar menu restoran
- [ ] Menerima pesanan pelanggan
- [ ] Menyimpan pesanan ke dalam keranjang
- [ ] Menghitung total biaya pesanan
- [ ] Menampilkan struk/receipt pesanan
- [ ] Opsi keluar dari program

### Struktur Program
- [ ] Menggunakan struktur keputusan (`if / elif / else`)
- [ ] Menggunakan struktur pengulangan (`while / for`)
- [ ] Menggunakan list/array
- [ ] Mengolah data string dengan benar

### Exception Handling
- [ ] Menangani input tidak valid
- [ ] Menangani menu yang tidak ditemukan
- [ ] Menangani pesanan kosong
- [ ] Program tidak langsung crash saat error

### Kelengkapan Pengumpulan
- [ ] Source code dikumpulkan dalam format PDF
- [ ] Video penjelasan tersedia
- [ ] Durasi video maksimal 15 menit
- [ ] Video menampilkan layar dan wajah
- [ ] Link video dapat diakses publik
